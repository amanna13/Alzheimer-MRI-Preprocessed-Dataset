## About the Dataset 
The data is collected from several websites, hospitals, and public repositories. The dataset consists of preprocessed MRI (magnetic resonance imaging) images. All the images are resized to 128 x 128 pixels. The dataset has four classes of images.
The Dataset consists of a total of 6400 MRI images. 

- Class - 1: Mild Demented (896 images)
- Class - 2: Moderate Demented (64 images)
- Class - 3: Non Demented (3200 images)
- Class - 4: Very Mild Demented (2240 images)
